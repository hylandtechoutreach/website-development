Final Project CSS

Final Project CSS

It's time to start adding some style to your final project, if you haven't already!

  1. Create a new style.css file in your HyTop project
  2. Add a new <link> element to your index.html file, within a <head> element
  3. Add some styles to the style.css file

The code should look something like this:

index.html

<link href="style.css" rel="stylesheet" type="text/css">

style.css

body {
  background: orange;
  color: blue;
  /* anything else ... */
}
